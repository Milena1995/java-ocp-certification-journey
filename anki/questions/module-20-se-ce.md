## Question

Complétez la phrase avec **se** ou **ce** :<br><br>

« Il ___ lave les mains avant de manger. »<br><br>

Please select 1 option<br><br>

A) ce<br>
B) se<br>
C) Les deux sont corrects<br>
D) Aucun des deux

## Réponse

<strong>Réponse correcte :</strong> B) se<br><br>

<strong>Règle fondamentale :</strong><br>
• <strong>se</strong> = pronom réfléchi → toujours placé devant un <strong>verbe pronominal</strong><br>
• Le verbe ici est <strong>se laver</strong> → pronom réfléchi obligatoire<br><br>

<strong>Astuce infaillible :</strong><br>
Remplacez le sujet par <strong>elle</strong> :<br>
→ « Elle <strong>se</strong> lave les mains » ✅ — la phrase reste naturelle avec <strong>se</strong><br>
→ Si ça fonctionne avec <em>se</em>, c'est <strong>se</strong>.<br><br>

<strong>A) est incorrect :</strong> « ce lave » n'a aucun sens — <em>ce</em> n'est pas un pronom réfléchi.<br>
<strong>C) est incorrect :</strong> seul <em>se</em> est grammaticalement possible ici.

## Question

Complétez la phrase avec **se** ou **ce** :<br><br>

« ___ livre est très intéressant. »<br><br>

Please select 1 option<br><br>

A) se<br>
B) ce<br>
C) Les deux sont corrects<br>
D) Aucun des deux

## Réponse

<strong>Réponse correcte :</strong> B) ce<br><br>

<strong>Règle :</strong><br>
• <strong>ce</strong> = déterminant démonstratif → remplace <em>un/le/mon</em> devant un nom<br>
• <strong>ce livre</strong> = « ce » détermine le nom « livre »<br><br>

<strong>Astuce :</strong><br>
Essayez de remplacer par <strong>le/la/mon/ma</strong> :<br>
→ « <strong>Le</strong> livre est très intéressant. » ✅ — fonctionne → c'est <strong>ce</strong><br><br>

Autres exemples de <strong>ce</strong> déterminant :<br>
• <em>ce matin</em>, <em>ce soir</em>, <em>cet homme</em> (devant voyelle), <em>cette femme</em> (féminin)<br><br>

<strong>A) est incorrect :</strong> « se livre » n'existe pas — <em>se</em> ne peut pas précéder un nom directement.

## Question

Laquelle de ces phrases utilise correctement **c'est** ou **s'est** ?<br><br>

Please select 2 options<br><br>

A) C'est une belle journée.<br>
B) S'est une belle journée.<br>
C) Il s'est trompé de chemin.<br>
D) Il c'est trompé de chemin.<br>
E) S'est difficile à comprendre.

## Réponse

<strong>Réponse correcte :</strong> A) et C)<br><br>

<strong>C'est vs S'est :</strong><br><br>

<strong>C'est</strong> = <em>cela est</em><br>
→ Remplacez par « <strong>cela est</strong> » : si ça marche → c'est <strong>c'est</strong><br>
→ « <strong>Cela est</strong> une belle journée. » ✅<br><br>

<strong>S'est</strong> = pronom <em>se</em> + auxiliaire <em>est</em> → toujours avec un verbe pronominal<br>
→ « Il <strong>s'est</strong> trompé » = « Il se tromper » au passé composé ✅<br><br>

<strong>B) est incorrect :</strong> « S'est une belle journée » — remplacer par « se est » n'a aucun sens.<br>
<strong>D) est incorrect :</strong> « c'est trompé » — on ne peut pas avoir <em>c'est</em> avec un verbe pronominal au passé composé.<br>
<strong>E) est incorrect :</strong> → « <strong>Cela est</strong> difficile » ✅ → il faut <em>c'est</em>.

## Question

Complétez avec **se** ou **ce** :<br><br>

« Je ne sais pas ___ qu'il veut. »<br><br>

Please select 1 option<br><br>

A) se<br>
B) ce<br>
C) Les deux selon le contexte<br>
D) Aucun des deux

## Réponse

<strong>Réponse correcte :</strong> B) ce<br><br>

<strong>Règle :</strong><br>
• <strong>ce que / ce qui</strong> = pronom démonstratif relatif (introduit une subordonnée)<br>
• « <strong>ce que</strong> » = la chose que → « Je ne sais pas <strong>ce que</strong> il veut. »<br><br>

<strong>Distinction :</strong><br>
• <strong>ce qui</strong> → sujet de la subordonnée → « <strong>Ce qui</strong> m'étonne, c'est ton calme. »<br>
• <strong>ce que</strong> → objet de la subordonnée → « Je vois <strong>ce que</strong> tu veux dire. »<br><br>

<strong>Astuce :</strong> « se que / se qui » n'existe pas en français — <em>se</em> ne peut jamais introduire une subordonnée.<br><br>

<strong>A) est incorrect :</strong> « se que » = construction inexistante en français.

## Question

Complétez les deux phrases avec **se** ou **ce** :<br><br>

1. « Elle ___ souvient de tout. »<br>
2. « ___ n'est pas ce que j'attendais. »<br><br>

Please select 1 option<br><br>

A) 1. ce — 2. se<br>
B) 1. se — 2. ce<br>
C) 1. se — 2. se<br>
D) 1. ce — 2. ce

## Réponse

<strong>Réponse correcte :</strong> B) 1. <strong>se</strong> — 2. <strong>ce</strong><br><br>

<strong>Phrase 1 :</strong> « Elle <strong>se</strong> souvient de tout. »<br>
→ Verbe pronominal : <em>se souvenir</em> → pronom réfléchi <strong>se</strong> ✅<br>
→ Test : « Elle <strong>se</strong> souvient » — remplacer le sujet ne change pas le pronom.<br><br>

<strong>Phrase 2 :</strong> « <strong>Ce</strong> n'est pas ce que j'attendais. »<br>
→ <strong>Ce</strong> = pronom démonstratif sujet → équivalent à « cela »<br>
→ Test : « <strong>Cela</strong> n'est pas ce que j'attendais. » ✅<br><br>

<strong>Résumé des tests :</strong><br>
• Remplacer par <strong>il/elle + se</strong> → pronom réfléchi → écrire <strong>se</strong><br>
• Remplacer par <strong>cela / le / la / mon</strong> → démonstratif → écrire <strong>ce</strong>

## Question

Laquelle de ces phrases est correcte ?<br><br>

Please select 1 option<br><br>

A) Il ce demande si c'est vrai.<br>
B) Il se demande si c'est vrai.<br>
C) Il se demande si s'est vrai.<br>
D) Il ce demande si s'est vrai.

## Réponse

<strong>Réponse correcte :</strong> B)<br><br>

<strong>Analyse :</strong><br><br>

<strong>« Il se demande »</strong> ✅<br>
→ Verbe pronominal : <em>se demander</em> → pronom réfléchi <strong>se</strong><br><br>

<strong>« si c'est vrai »</strong> ✅<br>
→ <strong>c'est</strong> = cela est → « si cela est vrai » ✅<br><br>

<strong>A) est incorrect :</strong> « ce demande » — <em>ce</em> ne peut pas être pronom réfléchi.<br>
<strong>C) est incorrect :</strong> « s'est vrai » — <em>s'est</em> nécessite un verbe pronominal au passé composé.<br>
<strong>D) est incorrect :</strong> Double erreur.

## Question

Complétez avec **se** ou **ce** :<br><br>

« ___ projet ___ déroule bien. »<br><br>

Please select 1 option<br><br>

A) se / ce<br>
B) ce / ce<br>
C) ce / se<br>
D) se / se

## Réponse

<strong>Réponse correcte :</strong> C) ce / se<br><br>

<strong>Phrase complète :</strong> « <strong>Ce</strong> projet <strong>se</strong> déroule bien. »<br><br>

• Premier blanc : <strong>Ce</strong> projet → <em>ce</em> est un déterminant démonstratif devant le nom « projet »<br>
→ Test : « <strong>Le</strong> projet se déroule bien. » ✅<br><br>

• Deuxième blanc : <strong>se</strong> déroule → verbe pronominal <em>se dérouler</em><br>
→ Test : « Elle <strong>se</strong> déroule bien. » ✅<br><br>

<strong>A) est incorrect :</strong> « Se projet » — <em>se</em> ne peut pas précéder un nom.<br>
<strong>B) est incorrect :</strong> « ce déroule » — <em>ce</em> n'est pas un pronom réfléchi.<br>
<strong>D) est incorrect :</strong> Double erreur.

## Question

Lequel des tests suivants permet de distinguer **se** de **ce** ?<br><br>

Please select 2 options<br><br>

A) Remplacer par « il/elle se » → si la phrase reste logique → écrire SE<br>
B) Remplacer par « cela » ou « le/la/mon » → si la phrase reste logique → écrire CE<br>
C) Si le mot est suivi d'un adjectif → toujours SE<br>
D) Si le mot est en début de phrase → toujours CE<br>
E) SE est toujours avant un nom ; CE est toujours avant un verbe

## Réponse

<strong>Réponse correcte :</strong> A) et B)<br><br>

<strong>Les deux tests fiables :</strong><br><br>

<strong>Test pour SE :</strong><br>
→ Changez le sujet : « il/elle <strong>se</strong> [verbe] » — si la phrase est naturelle → <strong>se</strong><br>
→ Ex : « Il <strong>se</strong> fatigue vite. » ✅<br><br>

<strong>Test pour CE :</strong><br>
→ Remplacez par « cela », « le », « la », ou « mon/ma » :<br>
→ Ex : « <strong>Ce</strong> résultat » → « <strong>Le</strong> résultat » ✅ → c'est <em>ce</em><br>
→ Ex : « <strong>Ce</strong> que tu dis » → « <strong>Cela</strong> que tu dis » ✅<br><br>

<strong>C) est incorrect :</strong> « Ce beau paysage » — <em>ce</em> peut précéder un adjectif + nom.<br>
<strong>D) est incorrect :</strong> « Se lever tôt est sain » — <em>se</em> peut être en début de phrase infinitive.<br>
<strong>E) est incorrect :</strong> C'est l'inverse : <em>CE</em> peut précéder un nom (déterminant), <em>SE</em> précède un verbe.
